TextProgress

TextProgress is a very simple text-based progress indicator. Insipored by terminal like proress bars it can be used to render simlpe progress.

Usage

Installation useing bower:

bower install --save text-progress
Or simply download text-progress.js and text-progress.css files and put them into your assets folder.

What it's not

TextProgress is not progress bar in the strict sense. It doesn't have methods for animation or nice transitions. All it does is provide textual rendering of the current state of the process, meaning reprecentation of the current value, process etc. out of the total.
